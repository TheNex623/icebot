# IceBot
Simple and crappy bot made by Mason Burdette.

## Setup/Installation

Download and Install [Node.js (LTS Version)](http://nodejs.org/)

- Windows
```sh
git clone https://github.com/zerowlan/icebot.git
cd icebot
npm install
```

- Mac or Linux
```sh
git clone https://github.com/zerowlan/icebot.git
cd icebot
sudo npm install
```
